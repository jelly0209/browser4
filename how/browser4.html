自作ブラウザの表示

1 ソースコード

今回vs codeで実際に書いていくコードです。名前はbrowser.pyで作成します。

2 importとfrom import

importはモジュールを使う際に必要な宣言です。
（モジュールとは.pyの拡張子で終わるpythonファイルを指す。）
importの種類は合計で3種類あります。

import A (モジュールAに定義されているすべての関数を読み込む)
import A as B (モジュールAをBとして読み込む（省略して宣言することが多い）)
from A import B (モジュールAの中にある関数やクラスBを読み込む)

今回のソースコードでは import ~ と from ~ importを使用します。
一番上のコードを例に出すと、sysで定義されている機能を使います!っていう感じです。

3 アプリケーションの起動

初めに my app = で自分で作成するアプリを定義します。
次に initurl = で起動時に開くURLを記述します。
（ここで記述するURLはGoogle でも safariでもegde でもなんでもいい。）

4 ブラウザの設定

起動するブラウザの設定をやっていきます。
browser = でブラウザを作成し、browser.load(QUrl(initurl))で作成したブラウザに表示するURLを設定します。
次にbrowser.resizeで表示するブラウザの初期サイズを設定します。
最後にbrowser.moveでブラウザの初期表示位置を設定します。（今回だと下に100,右に100の位置に表示します。）

5 ブラウザの表示と設定

browser.show()でブラウザを実際に表示させます。
最後にユーザがブラウザを閉じるとsys.exit(my_app.exec_())でブラウザが終了します。
